# Title

This a Tetris Game with JavaScript and HTML5.

# Install

```
git clone https://github.com/JohnGranger/teris_game.git
cd teris_game
python3 -m http.server
```

# Usage

```
cd teris_game
python3 -m http.server
```

Then open http://127.0.0.1:8000.
